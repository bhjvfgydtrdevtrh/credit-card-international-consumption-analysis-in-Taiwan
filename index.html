資料說明

本站分析數據整理自政府資料開放平台，統計期間自2014年起，包含台灣各主要地區以信用卡進行消費之相關數據，並著重於跨境交易部分。

資料來源： 政府資料開放平台 - 2014年至2025年之「六都十六縣居民跨境消費樣態」

指標計算方式與意義：

  • 平均跨境交易金額 (ATV): 跨境消費總金額 / 跨境消費總筆數

    意義：反映在跨境交易中，平均每一筆交易所花費的金額。此指標的變化可顯示消費單價的提升或下降趨勢，間接反映消費品質或購買商品類型的轉變。

  • 金額增長率: ((當期跨境總金額 - 前期跨境總金額) / 前期跨境總金額) * 100%

    意義：衡量特定期間內（例如年或月）跨境消費總金額相較於前一相同期間的變化百分比。正值表示增長，負值表示衰退，可用來判斷消費市場的擴張或萎縮速度。

  • 筆數增長率: ((當期跨境總筆數 - 前期跨境總筆數) / 前期跨境總筆數) * 100%

    意義：衡量特定期間內跨境交易總筆數相較於前一相同期間的變化百分比。此指標反映消費的活躍程度或頻率的變化。

  • ATV 增長率: ((當期ATV - 前期ATV) / 前期ATV) * 100% (若前期ATV為0，則視為無限大或特殊標記)

    意義：衡量特定期間內平均每筆跨境交易金額相較於前一相同期間的變化百分比。此為觀察「消費升級」現象的重要指標，若ATV增長率為正，可能代表消費者傾向購買更高單價的商品或服務。

  • 跨境消費金額佔比: (跨境交易金額 / 總交易金額) * 100%

    意義：反映在特定地區或時間段內，跨境消費金額在所有信用卡消費總金額（包含國內與跨境）中所佔的比重。此比例的變化可衡量該地區居民對海外商品或服務的依賴程度或偏好程度的演變。

  • 平均每卡跨境消費金額: 跨境交易金額 / 卡數

    意義：反映在特定期間內，平均每張流通的信用卡所產生的跨境消費金額。此指標可作為衡量單張卡片跨境消費貢獻度或消費強度的參考（請注意：本數據中的「卡數」可能為期間內的累計流通卡數，而非精確的當期活躍跨境消費卡數，解讀時需考量此差異）。
